# Pathfinding and Maze generation programm

A small programm that has animated pathfinding algorithms and is also able to generate mazes.

## Included Algorithms

### Pathfinding:
  * Dijkstra
  * Breath First Search
  * A-Star
	
### Maze generation:
  * Depth First Search
  * Prim
  
All the sprites were made by myself using Piskel.
